## Commits style
- 🎉 Initial commit
- 🐛 A bug fix
- ✨ A new feature
- 🔨 Doesn't fix a bug or add a function
- 🎨 Not affect the meaning of the code
- ⚙️ Add files and configurations
- 📝 Comments or readme notes

## Prettier format
```
npm i --global prettier
npx prettier --write .\src\
```
